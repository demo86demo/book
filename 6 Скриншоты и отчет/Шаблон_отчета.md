# Шаблон отчета

```text
ОТЧЕТ ПО ПРОВЕРКЕ РАБОТЫ ПРИЛОЖЕНИЯ «ЧИТАЙГОРОД»

1. Название проекта
Приложение для магазина книг «ЧитайГород».

2. Используемые технологии
Язык программирования: Python
СУБД: SQLite
Графический интерфейс: Tkinter

3. Реализованные модули
1) Авторизация пользователей
2) Просмотр списка товаров
3) Поиск, фильтрация и сортировка товаров
4) Работа с заказами
5) Добавление, редактирование и удаление товаров
6) Добавление, редактирование и удаление заказов
7) Работа с изображениями товаров

4. Проверенные сценарии
1) Вход в систему под разными ролями
2) Просмотр каталога товаров
3) Поиск товаров по текстовым полям
4) Фильтрация товаров по скидке
5) Сортировка товаров по цене и количеству
6) Добавление нового товара
7) Редактирование существующего товара
8) Удаление товара
9) Проверка запрета удаления товара, который входит в заказ
10) Просмотр и редактирование заказов
11) Проверка обработки ошибок ввода

5. Результат проверки
По итогам проверки основные функции приложения работают корректно.
Список товаров загружается из базы данных.
Поиск, фильтрация и сортировка выполняются корректно.
Операции добавления, редактирования и удаления товаров и заказов доступны согласно ролям пользователей.
Сообщения об ошибках выводятся в предусмотренных ситуациях.

6. Приложения
К отчету приложены скриншоты экранов, подтверждающие корректную работу программы.
```
