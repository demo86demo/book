# ЧитайГород

Настольное приложение на `Python + SQLite` для демонстрационного задания по модулям 1–3.

## Структура папок

- `1 ER и блок-схемы` — `er_diagram.pdf`, `algorithm.pdf`, файлы `drawio`
- `2 База Данных` — `bookstore.db` и `schema.sql`
- `3 Инструкция` — этот `README.md`
- `import` — исходные файлы импорта и ресурсы
- `5 Сборка для запуска` — код приложения и папка `images`
- `6 Скриншоты и отчет` — папка под скриншоты и итоговый отчет

## Запуск

```powershell
cd "5 Сборка для запуска"
python app.py
```

## Что делает приложение

- Использует данные из папки `import`
- Работает с базой `2 База Данных/bookstore.db`
- Хранит изображения товаров в `5 Сборка для запуска/images`
- Создает или обновляет схемы в папке `1 ER и блок-схемы`

## Тестовые учетные записи

- Клиенты создаются автоматически из импортированных заказов: `client1`, `client2` и далее, пароль `client123`
- Учетные записи менеджеров и администраторов импортируются из `import/user_import.xlsx`

Основные логины также отображаются на форме входа.

## Что нужно заскринить

Скриншоты лучше сохранять в папку `6 Скриншоты и отчет`.

- Окно входа с логотипом, полями логина и пароля, кнопкой входа и кнопкой входа как гость
- Вход в роль гостя и отображение списка товаров
- Вход в роль клиента и отображение ФИО пользователя в правом верхнем углу
- Вход в роль менеджера и работа поиска по товарам
- Работа фильтра скидки: `Все диапазоны`, `0-12.99`, `13-16.99`, `17+`
- Работа сортировки по цене
- Работа сортировки по количеству на складе
- Отображение товара со скидкой больше `25%` с цветовой подсветкой
- Отображение товара без остатка на складе с серой строкой
- Отображение перечеркнутой старой цены и итоговой цены
- Список заказов у менеджера или администратора
- Открытая форма добавления товара
- Открытая форма редактирования товара с заполненными полями
- Добавление нового товара администратором
- Редактирование существующего товара администратором
- Удаление товара, который можно удалить
- Сообщение об ошибке при попытке удалить товар, который есть в заказе
- Добавление или редактирование заказа администратором
- Сообщение об ошибке при неверном вводе данных
- Возврат на главный экран через кнопку `Выход`
